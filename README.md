# 学生信息管理系统

基于命令行的学生信息管理系统，使用 C99 开发。支持学生数据的增删改查、排序统计，数据自动持久化到文本文件。

## 环境要求

- 操作系统：Windows / Linux / macOS
- 编译器：GCC、Clang 或 CMake（任选其一）

## 编译

```bash
# 方式一：使用编译脚本（推荐，自动检测可用工具）
./compile.sh

# 方式二：手动编译
gcc -Wall -Wextra -std=c99 -o studentms src/*.c

# 方式三：CMake
mkdir build && cd build && cmake .. && make
```

编译产物为可执行文件 `studentms`。

## 运行

```bash
./studentms        # Linux / macOS
studentms.exe      # Windows
```

首次运行会自动创建 `students.txt` 数据文件。后续启动自动加载已有数据。

## 使用说明

启动后显示主菜单，输入数字选择操作：

```
+========================================+
|         学生信息管理系统 v1.0           |
+========================================+
|  1. 添加学生信息                        |
|  2. 显示所有学生信息                    |
|  3. 按学号查找学生                      |
|  4. 按学号删除学生                      |
|  5. 按学号修改学生信息                  |
|  6. 按成绩排序                          |
|  7. 统计信息                            |
|  8. 手动保存                            |
|  0. 退出系统                            |
+========================================+
请选择操作:
```

### 操作说明

| 选项 | 功能 | 说明 |
|------|------|------|
| 1 | 添加 | 依次输入学号、姓名、性别(男/女)、年龄、成绩 |
| 2 | 显示 | 表格形式列出所有学生 |
| 3 | 查找 | 输入学号查看该生详细信息 |
| 4 | 删除 | 输入学号删除记录 |
| 5 | 修改 | 输入学号后选择要修改的字段，回车保留原值 |
| 6 | 排序 | 选择升序或降序，按成绩排列 |
| 7 | 统计 | 显示总人数、平均分、最高/低分、不及格人数 |
| 8 | 保存 | 手动将内存数据写入文件 |
| 0 | 退出 | 自动保存后退出 |

### 输入规则

- **学号**：1 ~ 1999999999 之间的整数，不可重复
- **年龄**：0 ~ 150
- **成绩**：0.00 ~ 100.00
- 输入超长或非法字符时自动拒绝，不会导致程序异常

### 数据文件

数据存储在程序同目录下的 `students.txt`，每行一条记录：

```
学号 姓名 性别 年龄 成绩
1001 张三 男 20 88.50
```

可手动编辑该文件，但需确保每行恰好 5 个字段且数值合法，否则对应行会被自动跳过。
