# 学生信息管理系统

基于 C99 的命令行学生信息管理系统，采用动态链表 + 文本文件实现增删改查、排序、统计等核心功能。

## 快速开始

```bash
./compile.sh   # 编译（自动检测 CMake > GCC > Clang）
./studentms    # 运行
```

## 功能

| 功能 | 说明 |
|---|---|
| 添加 | 交互式输入学号/姓名/性别/年龄/成绩，自动校验唯一性 |
| 显示 | 表格形式列出全部记录，中英文混合列对齐 |
| 查找 | 按学号精确查找 |
| 删除 | 按学号删除，释放内存 |
| 修改 | 选择性修改，回车保留原值 |
| 排序 | 按成绩升序或降序（冒泡排序） |
| 统计 | 总人数、平均分、最高分、最低分、不及格人数 |
| 持久化 | 启动自动加载 `students.txt`，修改/退出自动保存 |

## 数据格式

`students.txt`，每行一条记录，空格分隔：

```
1001 张三 男 20 88.50
1002 李四 女 19 92.00
```

## 项目结构

```
src/
├── student.h    # 公共头文件（宏、结构体、函数声明）
├── main.c       # 入口 + 菜单调度
├── menu.c       # 菜单界面
├── student.c    # 链表操作（增删改查排序统计）
├── input.c      # 安全输入（fgets + strtol/strtof）
├── fileio.c     # 文件加载与原子保存
└── display.c    # CJK 宽度计算与列对齐
```

## 安全性

- **输入**：`fgets` 防缓冲区溢出，`strtol`/`strtof` 严格类型校验，学号/年龄/成绩硬范围限制
- **文件**：原子写入（备份 → 临时文件 → 替换），失败自动回滚
- **内存**：`free` 后头指针置 NULL，防悬空指针与 double-free
- **数据**：逐行解析 + 字段数校验 + 数值范围过滤，损坏行自动跳过
