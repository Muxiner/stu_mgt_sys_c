# 学生信息管理系统（设计文档）

基于 C99 的命令行学生信息管理系统。以动态单向链表为内存数据结构，以纯文本文件为持久化存储介质，实现学生信息的增删改查、排序、统计等核心功能。

## 项目结构

```
src/
├── student.h    # 公共头文件（宏、结构体、函数声明）
├── main.c       # 入口 + 菜单调度
├── menu.c       # 菜单界面
├── student.c    # 链表操作（增删改查、多字段排序、统计、释放）
├── input.c      # 安全输入（fgets + strtol/strtof）
├── fileio.c     # 文件加载（尾插法）与原子保存
└── display.c    # CJK 显示宽度计算与列对齐
```

## 技术栈

| 技术 | 用途 |
|------|------|
| C99 | 全部业务逻辑、数据结构、文件 I/O、内存管理 |
| 动态单向链表 | `struct Student` + `next` 指针，无最大人数限制 |
| 文件 I/O | `fgets`+`sscanf` 加载，`fprintf` 保存，字段以空格分隔 |
| 内存管理 | `malloc` / `free` 动态管理结点，头指针释放后置 NULL |
| 宽字符处理 | `mbstowcs` + `wcswidth` 计算 CJK 字符终端列宽 |

## 功能

| 功能 | 说明 |
|---|---|
| 添加 | 交互式输入，自动校验学号唯一性 |
| 显示 | 表格形式列出全部记录，中英文混合列对齐 |
| 查找 | 按学号精确查找，显示单条完整信息 |
| 删除 | 按学号删除结点，释放内存，自动保存 |
| 修改 | 选择性修改，回车保留原值 |
| 排序 | 支持按学号/姓名/年龄/成绩，升序/降序（冒泡排序交换结点指针） |
| 统计 | 总人数、平均分、最高分、最低分、不及格人数 |
| 持久化 | 启动加载 `students.txt`，修改/退出自动保存 |

## 数据格式

`students.txt`，每行一条记录，空格分隔：

```
学号 姓名 性别 年龄 成绩
1001 张三 男 20 88.50
1002 李四 女 19 92.0
```

## 安全性

| 维度 | 措施 |
|------|------|
| 输入 | `fgets` 防缓冲区溢出；`strtol`/`strtof` 严格类型校验；学号/年龄/成绩硬范围限制；超长输入自动清空 stdin 残留 |
| 文件 | 原子写入（备份 → 临时文件 → 替换），写入失败自动回滚，原数据不丢失 |
| 加载 | 尾插法保持文件顺序；逐行解析 + 字段数校验 + 数值范围过滤，损坏行跳过 |
| 内存 | `free` 后头指针置 NULL，防悬空指针与 double-free |
| 交互 | 空输入拒绝（修改模式除外），纯空白等价于空输入 |
